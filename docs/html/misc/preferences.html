Preferences in veefireÂ¶

Preferences for veefire are kept in an xml-file, preferences.xml.

Structure - the Preferences logic:

<preferences>
<name-of-property value=”propertyvalue” /> api.dbapi.Preferences ( current preference )
<options>
<name-of-property> api.dbapi.Preferences ( preference )
<option value=”optionvalue”> ( preference option )
<option value=”optionvalue2”> ( preference option )
</name-of-property>

Example:
